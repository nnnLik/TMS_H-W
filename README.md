# **STEPS**

* create rep on [GitHub](https://github.com/)
* create local rep
## *in local rep v1*
1. git init
2. git status
3. create .gitignore
4. git add.
5. git commit -m 'first commit'
6. git remove add origin http://....
7. git push -u origin master

## *in local rep v2*
1. delete .git dir
2. git clone [ссылка на репозиторий]
3. copy .git in project
4. git status
5. git add .
6. git commit -m ''
7. git push
8. loggin-password(token)
